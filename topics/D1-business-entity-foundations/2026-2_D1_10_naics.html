NAICS Codes

Six-digit industry classification system used for internal census purposes
What It Is

North American Industry Classification System (NAICS)

The NAICS code is a six-digit number that classifies businesses according to their primary industry or activity. It is used by government agencies and businesses to collect economic data, analyze industry trends, and support marketing efforts. Importantly, the NAICS code does not affect any legal filings or company information.

Customers are required to provide a NAICS code during the order process in states where this information is recorded. In states where it is not mandatory, customers will still be prompted to select a NAICS code after placing their order and before they can access their company dashboard.

Rules
1
Permitted

Agents may share general information and direct customers to www.census.gov/naics/ to help them choose the appropriate code.

2
Not Permitted

Customers must enter the NAICS code themselves. Agents should not input it on their behalf.

3
Code Updates

If not required by the state, the NAICS code can be updated internally by requesting the new preferred code through email.
