What It Is

If a customer wants to switch their Registered Agent, let them know this requires filing a Change of Agent form with the state. This updates the agent's name, address, or contact information in the public record. We offer this service in all 50 states.

We charge a $99 processing fee + state fee to file a Change of Agent on behalf of our customers.

3.2 Important Considerations

Separate Filing Required

The Registered Agent can't be changed through an Annual Report or Amendment. It must be done through a dedicated Change of Agent filing.

RA Doesn't Cancel Automatically

If a customer changes their agent, our service doesn't automatically cancel. They'll need to let us know if they want to stop using our Registered Agent service.

Free First Year Only Applies as Part of a Business Formation Package

The first year of our Registered Agent service is only free when they form a business with us. Regular pricing applies if purchasing the service separately.

Public Info

The name and address of the Registered Agent are always publicly visible. If a customer wants to protect their privacy, recommend using our service instead of listing a personal address.

3.3 Customer View

Here's how Registered Agent Services will appear in a customer's dashboard.

New Registered Agent Service

This option is for customers who want to start using Bizee as their Registered Agent. If they're not currently working with us and are looking to appoint Bizee for the first time, this is the correct path.

Change Registered Agent

This is for customers who are already using Bizee as their Registered Agent but now want to switch to a different provider. Selecting this allows them to formally notify the state and make that transition.

Helpful Tip: If a customer is transferring their registered agent to or from Bizee, be sure they click "Yes" when asked if they'd like help with the filing. This ensures that the Change of Agent filing is handled correctly and helps avoid delays or complications in the process. Being familiar with these labels in the dashboard helps keep the process smooth, clear, and accurate for both you and the customer.
