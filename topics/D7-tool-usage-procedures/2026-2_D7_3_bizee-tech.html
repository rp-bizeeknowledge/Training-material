Bizee Tech

Order search · Order detail view · DB comments
What It Is

Bizee Tech, formerly named DB, is Bizee's internal database system and the central operational platform used by all involved departments to manage customer accounts. It stores client information, order processing data and automations, serving as the record system for all company and order-related activity.

In addition to providing visibility into company details, DB/Bizee Tech functions as an internal search tool that allows users to locate, review, and document information across accounts. It is also the system through which agents take action on accounts, including processing cancellations, issuing refunds, and updating order statuses.

Because actions taken in DB/Bizee Tech directly affect order outcomes and financial transactions, accuracy and careful review are essential. Agents should rely on KB when providing assistance to customers or performing account-level actions, ensuring that all activity aligns with established procedures and policies.

Order Search View

Order Search is DB/Bizee Tech's main page. From it, you will be able to pull up the accounts that you'll be working with when assisting a customer. You will also be able to look for orders that match certain criteria by filtering them with the dropdown menus available on the top section.

Searchbar-Supported Criteria

These is the data types you can use to pull up an account:

The phone number associated with the account
The order number
The email address linked to the account
The company name
The name of the authorized user
Order Detail View

The order detail view will open once you click on an order in the order search page. On this page, all the details regarding the order will appear, and it will be possible for you to manage certain aspects or information of the order.
